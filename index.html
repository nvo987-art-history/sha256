SHA256 GENERATOR

INPUT: Text or File
FEATURES: SHA-256 hashing, ZIP export

IMPORTANT:
Text input is encoded as UTF-8.
File hashing is performed on raw binary data.
Results may differ for the same visible content.

INFO:
Processing is performed locally in your browser.
No input data is transmitted to any server.
